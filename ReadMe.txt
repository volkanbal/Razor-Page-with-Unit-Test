This solution is a demo for ticket service. It has a public ticket form for contacting Customer Support. 
* The form has several fields.
* The data is  validatiing with data annotation with regex expressions.
* It use Entity Framework in memory database to keep data.
* Frontend uses jquery and AdminLte 3 responsive Interface
* Unit test is implemented for testing.

You can open the solution by ArvatoApp\ArvatoApp.sln
